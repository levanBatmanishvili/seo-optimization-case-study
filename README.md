# SEO Optimization Case Study

## Project Overview
This project focuses on technical SEO optimization and performance improvement of an existing photography website.

The goal was to improve:
- Page speed
- SEO structure
- Accessibility
- Lighthouse performance score

---

## Initial Audit
An SEO and performance audit was conducted using Lighthouse and Chrome DevTools.

### Issues Found
- Missing meta descriptions
- Incorrect heading structure (H1, H2, H3)
- Images without alt attributes
- Large images slowing down the website
- Unoptimized CSS and JavaScript
- Poor Lighthouse performance score

---

## Optimizations Implemented
The following optimizations were implemented:

- Added meta titles and meta descriptions
- Fixed heading hierarchy (H1 → H2 → H3)
- Added alt attributes to all images
- Converted images to WebP format
- Compressed and resized images
- Minified CSS and JavaScript
- Improved accessibility
- Improved page loading performance

---

## Results

### Lighthouse Score Improvement

| Metric | Before | After |
|-------|-------|------|
| Performance | 58 | 92 |
| SEO | 70 | 100 |
| Accessibility | 80 | 95 |
| Best Practices | 75 | 100 |

The website performance and SEO score improved significantly after optimization.
