# SEO Optimization Case Study

## Project Overview
Technical SEO optimization and performance improvement of a website.

## SEO Audit
- Missing meta descriptions
- Incorrect heading structure
- Images without alt attributes
- Low Lighthouse score
- Large images slowing down the website

## Optimizations
- Added meta tags
- Fixed heading structure
- Added alt attributes
- Optimized images
- Improved performance

## Results
Lighthouse score improved:

| Metric | Before | After |
|-------|-------|------|
| Performance | 58 | 92 |
| SEO | 70 | 100 |
| Accessibility | 80 | 95 |

## Tools
- Lighthouse
- Chrome DevTools
- Google Search Console
